### [🤖 Sentinel AI](https://github.com/Soulcynics404/Sentinal-AI)
**Continuous Face Verification Security System**
`C++17` `OpenCV` `dlib`

Engineered a real-time face verification system using OpenCV and dlib ResNet 128D embeddings that monitors the laptop owner's face and **auto-locks within 3 seconds** when an unauthorized user or absence is detected.

Includes camera tamper detection (blur, dark, covered), smart multi-person alert, and a kill switch with 3 hash-verified security questions for identity recovery.

Integrated a **Telegram Bot with 27+ remote commands** for lock/unlock, live camera capture, video recording, and CSV activity log export with millisecond-precision timestamps across 16 data fields.

**Web Vulnerability Scanner & VAPT Reporter**
`Rust` `Tokio` `Async`

Architected from scratch in Rust with a **5-phase automated pipeline** — reconnaissance, fingerprinting, CVE intelligence, active scanning, and reporting — with 11 custom detection modules for SQLi, XSS, SSRF, LFI, SSTI, CORS, CSRF, and SSL/TLS.

Connected **NVD API 2.0, ExploitDB, EPSS scores, and CISA KEV catalog** for live CVE lookup with real CVSS scoring and exploit availability data.

Generates full VAPT reports in **4 formats (HTML, JSON, PDF, SARIF)** with exact payload highlighting and step-by-step remediation guidance.
